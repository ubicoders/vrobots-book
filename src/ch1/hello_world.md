## List the topics from Virtual Robots

Try this in the terminal.

```
list-topics
```

This will output the below.

```
Listing topics...
  - [i] vr/0/cams/down/720p
  - [i] vr/0/cams/left/720p
  - [i] vr/0/cams/right/720p
  - [z] vr/0/states
```

The sim publishes the topics like this.
